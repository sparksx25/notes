# Windows VSCODE Shortcut Key

## 转到菜单
- 向下插入一行
    `CTRL+Enter`

- 向上插入一行
    `CTRL+SHIFT+Enter`

- 向下复制一行
    `CTRL+ALT+downArrow`

- 向上复制一行
    `CTRL+ALT+upArrow`
    
- 缩进当前行
    `CTRL+[`

- 后退当前行
    `CTRL+]`

- 删除当前行
    `CTRL+SHIFT+K`

- 折叠全部行
    `CTRL+K CTRL+O`

- 展开全部行
    `CTRL+K CTRL+J`

- 折叠当前块
    `CTRL+K CTRL+[`

- 展开当前块
    `CTRL+K CTRL+]`

- 转到匹配的括号
    `CTRL+SHIFT+\`

- 在当前文件种查找
    `CTRL+F`

- 转到文件: 根据文件名搜索搜索文件
    `CTRL+P`

- 转到行列
    `CTRL+G`, 如 `:4:1` 第四行，第一个字符

## 查看菜单
- 切换资源管理器
    `CTRL+B`

- 预览 MD
    `CTRL+SHIFT+V`

- 打开工作区搜索面板
    `CTRL+SHIFT+F`

- 打开工作区拓展面板
    `CTRL+SHIFT+X`

- 打开命令行面板: 执行vscode命令
    `CTRL+SHIFT+P`
