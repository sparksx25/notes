# 术语

## DAO(data access object：数据访问对象)
- 负责像数据库写数据，从数据库读数据

## POJO(plain ordinary java object: 普通 java 对象)
普通的 java 对象，只负责存取数据，没有其他的操作，如以下对象
- DTO: 接收前端接口数据的类
- VO: 返回给前端数据的类
- BO: 业务层数据
