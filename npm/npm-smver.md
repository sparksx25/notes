# npm 的语义化版本

## Ranges
`>=1.2.7<1.3.0` 表示

## Advanced Range Syntax
Hyphen Ranges `X.Y.Z - A.B.C`
- `1.2.3 - 2.3.4` := `>=1.2.3 <=2.3.4`
- `1.2.3 - 2` := `>=1.2.3 <3.0.0` 

X-Ranges 
- `1.2.x` := `>=1.2.0 <1.3.0`
- `1.*` := `>=1.0.0 <2.0.0`

Tilde Ranges
- `~1.2.3` := `>=1.2.3 <1.(2+1).0` := `>=1.2.3 <1.3.0`
- `~0.2.3` := `>=0.2.3 <0.(2+1).0` := `>=0.2.3 <0.3.0`

Caret Ranges
- `^1.2.3` := `>=1.2.3 <2.0.0` 
- `^0.2.3` := `>=0.2.3 <0.3.0`

- 摘自[npm 版本范围表示](https://docs.npmjs.com/cli/v6/using-npm/semver)
